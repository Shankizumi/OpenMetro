                                               Weather Now

Weather Now is a simple weather application designed to help outdoor enthusiasts like Jamie quickly check the current weather conditions for any city. This app fetches real-time weather data using the Open-Meteo API and displays it in a clean, user-friendly interface.

                                                Features
*Real-time weather information for any city.
*Minimalist and responsive design for both desktop and mobile users.
*Intuitive user interface for easy navigation.

                                            Technologies Used
*HTML5 and CSS3: For the structure and styling of the application.
*JavaScript: For making API requests and rendering data dynamically.
*Open-Meteo API: To fetch real-time weather data.
*Responsive Design Techniques: Ensuring the application looks good on all devices.

*Open the Weather Now application in your web browser.
*Enter the name of any city in the search bar.
*View the current weather details, including temperature, wind speed, and more.

                                            API Information
*Open-Meteo API is a free-to-use weather data provider. The API documentation can be found at Open-Meteo API Docs.
*How We Use It
    The app sends a GET request to the Open-Meteo API, passing the city name as a parameter.
    The response data is parsed and displayed in an easy-to-read format.
